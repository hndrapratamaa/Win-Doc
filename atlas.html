4. Instalasi Atlas OS

Mengubah instalasi Windows bersih menjadi sistem operasi super ringan, responsif, dan minim *latency*.

üîß Mekanisme & Prasyarat Penting

AtlasOS adalah **Playbook** yang dijalankan melalui AME Wizard. Fungsinya menghapus *bloatware*, mematikan *telemetri*, dan menonaktifkan layanan tidak penting untuk memaksimalkan performa. Atlas **mensyaratkan instalasi Windows yang bersih dan lengkap** (dari Bab 2) untuk menjamin stabilitas.

‚ö†Ô∏è Peringatan:
  • AtlasOS **mewajibkan instalasi Windows bersih** (lihat Bab 2).
  • **Tidak dapat di-uninstall** tanpa melakukan *reinstall* Windows.
  • Edisi **Windows 11 Pro** sangat direkomendasikan (Home tidak didukung resmi).

üì¶ Unduhan Resmi:

  • **AME Wizard Beta**
  • **Atlas Playbook (.apbx)**
Unduh di Situs Resmi AtlasOS

üìù Bagian A: Persiapan & Update (WAJIB)

Sebelum menjalankan Playbook, Windows harus berada dalam kondisi *fully updated* agar proses optimasi stabil.

  1. Download dan ekstrak AME Wizard & Atlas Playbook.
  2. **Pastikan driver jaringan telah terinstal** (lihat Bab 2).
  3. Buka **Settings ‚Üí Windows Update**. Install semua *update* (termasuk *optional updates*).
  4. Buka **Microsoft Store ‚Üí Library** dan *Update All*.
  5. **Restart** perangkat setelah semua *update* selesai. Ulangi pengecekan *update* sekali lagi untuk memastikan sistem benar-benar siap.

üöÄ Bagian B: Penerapan Playbook

  1. Jalankan **AME Wizard Beta.exe** sebagai Administrator.
  2. *Drag & drop* file **Atlas Playbook.apbx** ke jendela AME Wizard.
  3. Ikuti seluruh instruksi pada layar, berikan persetujuan untuk perubahan permanen seperti Penonaktifan Windows Defender, Telemetri, dan penghapusan aplikasi bawaan.
  4. Sistem akan melakukan beberapa proses otomatis dan *restart* beberapa kali.

üìâ Daftar Perubahan Sistem oleh AtlasOS

Atlas akan menghapus/menonaktifkan komponen untuk performa ekstrem. Harap ketahui daftar perubahan ini:

Komponen yang Dihapus / Dinonaktifkan Permanen

  • **Smart App Control** (Dihapus karena dapat menyebabkan *slowdown*).
  • **Aplikasi Store bawaan** (Teams, Xbox, Clipchamp, dll.).
  • **Semua jenis Telemetry** dan fitur *Windows tips*.
  • **Automatic Windows Update** dan **Windows Defender** (dinonaktifkan untuk menghemat RAM).

üí° Pasca Instalasi & Hasil Akhir:

  • **Hapus Windows.old:** Bersihkan sisa file instalasi lama melalui Settings ‚Üí Storage.
  • **Instal Driver Manual:** Wajib menginstal driver GPU dan Chipset secara manual dari situs produsen.
  • **Hasil Akhir:** RAM *idle* akan berada di sekitar **1.0 ‚Äì 1.3GB**, menjadikan PC sangat responsif untuk perangkat low-end.
