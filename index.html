1. Pendahuluan

Mengapa Windows 11 di Laptop/Pc Low-end Perlu Optimasi.

Tujuan Utama Dokumentasi

Dokumentasi Win-Doc ini disusun khusus untuk pengguna laptop atau PC dengan spesifikasi terbatas, terutama RAM 4GB atau penyimpanan HDD lama. Tujuannya adalah membuat Windows 11 menjadi lebih ringan dan responsif.

Panduan mencakup instalasi bersih, optimasi sistem, dan pemilihan aplikasi ringan untuk kebutuhan harian.

Tantangan RAM 4GB

Windows 11 secara default menggunakan sekitar 2.5GB hingga 3GB RAM saat idle. Optimasi dalam dokumentasi ini bertujuan menurunkan penggunaan hingga di bawah 1.5GB.

⚠️ Backup data sebelum memulai karena proses melibatkan instalasi ulang sistem.

Persiapan

  • Flashdisk minimal 8GB
  • Koneksi internet stabil
  • Perangkat terhubung charger
  • Lisensi Windows valid

Risiko

  • Beberapa fitur Windows dapat nonaktif
  • Fokus pada performa bukan kompatibilitas penuh
  • Disarankan menggunakan Windows Pro atau IoT
