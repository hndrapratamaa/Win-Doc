Persiapan Clean Install

‚ö†Ô∏è Backup Data Penting

Proses *Clean Install* akan menghapus seluruh data pada partisi sistem (Drive C). Harap **pastikan Anda telah mem-backup semua file penting** dari folder C:\Users/[Username] (Desktop, Documents, Downloads, Pictures) ke partisi lain (D:) atau penyimpanan eksternal.

Cek Status Layanan Windows (Deteksi Virus):

Jika sistem lama Anda dicurigai terinfeksi virus, virus dapat menyebar ke installer USB. Jalankan perintah ini di PowerShell Admin untuk mengecek status layanan penting:

sc.exe start "sppsvc" > $null 2>&1; Write-Host "Error code: $LASTEXITCODE"

Jika output adalah **577** (Windows Defender dinonaktifkan secara permanen) atau **225** (Error file/layanan), sistem Anda mungkin terinfeksi. Gunakan PC lain untuk membuat USB *bootable* jika terjadi error.

‚ö†Ô∏è Peringatan: Persiapan Driver Jaringan

Setelah *Clean Install*, Windows mungkin tidak memiliki driver jaringan/Wi-Fi secara *default* saat pertama kali masuk ke *desktop*. Untuk mengatasi hal ini, Anda sangat disarankan untuk:

  • **Download driver jaringan** (Wi-Fi dan Ethernet) PC/Laptop Anda dari situs resmi produsen (HP, Lenovo, ASUS, dll.) *sebelum* proses instalasi.
  • Simpan file driver tersebut di Flashdisk atau penyimpanan eksternal yang mudah diakses.
  • Ini memastikan Anda bisa langsung terkoneksi internet untuk langkah aktivasi dan update selanjutnya.

Membuat Bootable USB (Rufus)

Gunakan aplikasi Rufus untuk membuat installer dengan kemampuan *bypass* persyaratan hardware.

  • Colokkan Flashdisk (Perhatian: Semua data di dalamnya akan dihapus).
  • Buka Rufus, pilih ISO Windows 11 Anda.
  • Klik START. **Pastikan Anda mencentang opsi** "Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0" & "Online Microsoft account".
üõ†Ô∏è Opsi Advanced: Install Edisi IoT Enterprise

Untuk menginstal edisi **IoT Enterprise** secara paksa, yang merupakan versi paling ringan:

Di dalam Flashdisk Installer yang sudah dibuat Rufus, buka folder \sources\, buat file bernama **PID.txt**, dan isi dengan kode berikut:

Untuk IoT Enterprise GAC (Standard):

[PID] Value=XQQYW-NFFMW-XJPBH-K8732-CKFFD

Untuk IoT Enterprise LTSC (Jangka Panjang):

[PID] Value=CGK42-GYN6Y-VD22B-BX98W-J8JXD

File PID.txt ini akan memaksa Setup memilih edisi yang Anda tentukan saat instalasi.

üìã Opsi Advanced: Munculkan Semua Pilihan Edisi (ei.cfg)

Jika laptop Anda memiliki lisensi bawaan (OEM) di BIOS (misalnya Windows Home), installer akan otomatis memilih edisi tersebut. Untuk memunculkan semua pilihan edisi (Home, Pro, Enterprise, dll.):

Di folder \sources\ pada flashdisk, buat file **ei.cfg** dengan isi:

[Channel] NoKeyChannel

**Catatan:** Pilih opsi "Previous version of setup" saat booting installer (jika muncul) agar trik ini bekerja di Windows 11 versi terbaru (24H2+).

Proses Clean Install

Langkah krusial untuk memastikan sistem benar-benar bersih dan ringan:

  • Cabut semua *harddisk* eksternal atau flashdisk lain (hanya sisakan Flashdisk Installer) untuk keamanan data.
  • Restart PC, tekan tombol **Boot Menu** (biasanya F11, F12, Esc, atau Del sesuai merk PC Anda).
  • Pilih Flashdisk USB Anda di menu *Boot*.
  • Pilih "Custom: Install Windows only" saat ditanya jenis instalasi.
  • **Hapus semua partisi** yang berkaitan dengan sistem lama (Drive C: System, Reserved, Primary) hingga hanya tersisa **"Unallocated Space"** (ruang yang tidak teralokasi). Kemudian, klik **Next**. *Ini penting untuk menghilangkan sisa-sisa bloatware lama.*

Tips: Bypass Syarat Internet & Akun Microsoft (OOBE)

Jika Anda tidak menggunakan opsi *bypass* Rufus dan terhenti di layar "Let's connect you to a network", ikuti langkah manual ini untuk memaksa masuk ke mode *Local Account*:

  1. Tekan tombol **Shift + F10** untuk membuka *Command Prompt*.
  2. Ketik perintah ini dan tekan Enter:
start ms-cxh:localonly

Komputer akan *restart* ke mode OOBE (Out-of-Box Experience) dan opsi untuk membuat akun lokal (*offline account*) akan muncul.

Troubleshooting Instalasi

  • **Error 'PC doesn't meet the minimum requirements':** Cek ulang bahwa Anda telah mencentang opsi *bypass* di Rufus, atau pastikan TPM 2.0 dan Secure Boot sudah di-enable di BIOS/UEFI.
  • **Error Code 0xC1900101 (atau sejenisnya):** Umumnya masalah driver atau *hardware* yang tidak kompatibel. Pastikan driver GPU/Chipset sudah di-*update* ke versi stabil.
  • **Setup keluar tiba-tiba:** Cek kesehatan RAM (*Memory Diagnostic*) dan pastikan flashdisk Anda tidak rusak.

Prosedur Rollback & Pemulihan

Karena langkah optimasi bersifat permanen dan agresif, **sangat disarankan** Anda menguasai cara pemulihan sistem:

  • **Backup Image Sistem:** Kami merekomendasikan pembuatan **System Image** dengan alat seperti Macrium Reflect (sebelum instalasi) untuk *Rollback* cepat.
  • **Pemulihan Sistem:** Jika hanya terjadi error kecil (*service* mati, *file* rusak), buka **PowerShell Admin** dan jalankan perintah perbaikan sistem: 
    sfc /scannow && DISM /Online /Cleanup-Image /RestoreHealth
  • **Rollback Total:** Jika semua gagal, Anda harus mengulang proses **Clean Install** dari Bab 2.
