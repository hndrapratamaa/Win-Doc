5. Manual Tweaking

Optimasi manual jika Anda memilih tidak menggunakan Atlas OS.

Debloating Cepat dengan Script

Jika merasa Atlas OS terlalu ekstrem, Anda bisa menggunakan Chris Titus Tech Tool. Alat ini berbasis PowerShell dan menyediakan menu interaktif untuk melakukan *debloating* yang lebih moderat:

  1. Buka **PowerShell sebagai Administrator**.
  2. Salin dan jalankan perintah ini:
iwr -useb https://christitus.com/win | iex
  • Di tab "Tweaks", pilih profil **"Desktop"** (untuk penggunaan standar) atau **"Laptop"**.
  • Klik **"Run Tweaks"**. Proses ini akan mematikan Telemetry, menghapus Temporary files, dan menerapkan optimasi ringan lainnya.

Matikan Service Berat (Services.msc)

Menonaktifkan layanan Windows yang tidak perlu adalah kunci untuk mengurangi penggunaan RAM dan CPU di latar belakang. Tekan **Win + R**, ketik **"services.msc"**, lalu cari dan ubah **"Startup type"** menjadi **"Disabled"** untuk layanan berikut:

  • **Connected User Experiences and Telemetry:** Wajib dimatikan. Bertanggung jawab atas pengiriman data ke Microsoft.
  • **SysMain (Dulu Superfetch):** Matikan jika PC/Laptop Anda menggunakan **SSD**. Jika Anda masih menggunakan HDD jadul, biarkan saja (membantu *boot time*).
  • **Windows Search:** Matikan jika Anda sering mengalami *disk usage* 100% atau jika Anda jarang menggunakan fitur pencarian Windows.
  • **Downloaded Maps Manager:** Jika Anda tidak pernah menggunakan aplikasi Maps bawaan Windows, layanan ini tidak diperlukan.

Optimasi Visual Effects

Windows menggunakan sumber daya GPU/CPU untuk animasi dan efek visual. Menonaktifkannya dapat memberikan peningkatan performa yang signifikan. Caranya:

  • Ketik **"View advanced system settings"** di *Start Menu*.
  • Pilih tab **Advanced** lalu klik tombol **Settings** di bawah *Performance*.
  • Pilih **"Adjust for best performance"** (ini akan membuat tampilan Windows menjadi "burik").
  • Untuk tampilan yang lebih baik, centang kembali:
    • "Show thumbnails instead of icons"
    • "Smooth edges of screen fonts"
