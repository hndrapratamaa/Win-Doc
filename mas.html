3. Aktivasi (MAS) Updated Nov 2025

Metode Aktivasi Permanen (HWID/OHook) dan KMS untuk Windows & Office.

⚠️ Legal & Safety Disclaimer

Panduan aktivasi ini hanya menjelaskan konsep dan mekanisme dari metode komunitas. **Selalu** gunakan lisensi resmi Microsoft (retail, OEM, atau Volume Licensing) yang sah.

Penulis/Dokumen ini **tidak menganjurkan pembajakan** dan **tidak bertanggung jawab** atas segala konsekuensi hukum atau teknis dari penggunaan alat pihak ketiga.

Opsi Legal: Kunjungi [Situs Microsoft Volume Licensing] atau beli lisensi dari reseller resmi.

Metode 1: PowerShell (Recommended) ❤️

1. Klik kanan tombol Start (Win + X), pilih Terminal (Admin) atau PowerShell (Admin).

2. Salin dan tempel perintah di bawah ini, lalu tekan Enter:

irm https://get.activated.win | iex

Metode ini berfungsi untuk Windows 8/10/11. Menu aktivasi akan muncul, pilih opsi 1 (HWID) untuk aktivasi permanen.

Jika perintah di atas Error / Diblokir ISP (Menggunakan DNS-Over-HTTPS):

Gunakan perintah alternatif ini (membutuhkan Windows 10/11 versi terbaru):

iex (curl.exe -s --doh-url https://1.1.1.1/dns-query https://get.activated.win | Out-String)

Metode 2: Traditional (Manual)

Jika Terminal/PowerShell diblokir atau terjadi error yang tidak teratasi, gunakan metode manual:

  • Kunjungi repository resmi MAS dan unduh file MAS_AIO.cmd.
  • Klik kanan file tersebut, lalu pilih Run as Administrator.
  • Ikuti petunjuk di layar (Pilih opsi 1 untuk HWID).

Upgrade Edisi Windows (Penting untuk Optimasi)

Jika Anda menginstal Windows Home / Single Language, sangat disarankan untuk meng-upgrade ke edisi **Pro** atau **Enterprise**. Gunakan fitur "Extras" atau "Change Windows Edition" di dalam menu MAS untuk melakukan *upgrade*.

Upgrade ini krusial karena edisi Pro/Enterprise memungkinkan Anda mengakses **Group Policy Editor (gpedit.msc)**, yang merupakan alat utama untuk menonaktifkan Telemetry dan layanan latar belakang yang berat.

Mini FAQ Aktivasi

Q: Apa bedanya HWID dan Online KMS?

A: **HWID** (*Hardware ID*) memberikan aktivasi permanen yang tersimpan di server Microsoft. Selama Anda tidak mengganti motherboard, aktivasi akan otomatis berlaku bahkan setelah instalasi ulang. **Online KMS** hanya memberikan lisensi selama 180 hari, dan sistem perlu melakukan re-aktivasi (perpanjangan) setelah periode tersebut berakhir.

Q: Jika HWID gagal, apa langkah selanjutnya?

A: HWID biasanya hanya berlaku jika sistem Anda pernah teraktivasi secara permanen sebelumnya. Jika gagal, coba gunakan opsi Digital License (jika ada) atau gunakan metode Online KMS sebagai alternatif untuk sementara.

Q: Perlu uninstall Office lama sebelum aktivasi OHook?

A: Tidak wajib, tetapi sangat disarankan untuk melakukan *Clean Install* Office menggunakan metode ODT (Bab 6) untuk memastikan tidak ada konflik lisensi atau sisa file lama yang menyebabkan kegagalan OHook.
