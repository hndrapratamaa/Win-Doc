6. Microsoft Office (ODT)

Cara resmi & paling ringan install Office menggunakan Office Deployment Tool.

Apa itu ODT?

Office Deployment Tool (ODT) adalah alat resmi Microsoft yang memungkinkan kita mengunduh dan menginstal versi Office (Click-to-Run) dengan konfigurasi yang bisa disesuaikan (misalnya hanya Word, Excel, PowerPoint). Metode ini jauh lebih **bersih, cepat, dan ringan** dibandingkan installer standar.

Catatan: Konfigurasi yang kita gunakan (Channel="PerpetualVL2021") adalah untuk lisensi Volume (Office 2021), yang kompatibel dengan metode aktivasi KMS (seperti OHook di Bab 3).

Langkah 1: Persiapan Bahan

  1. Download **Office Deployment Tool** dari situs resmi Microsoft.
  2. Buat folder baru di Local Disk C, beri nama Office (Jalur: C:\Office).
  3. Jalankan file ODT yang didownload, **ekstrak isinya** ke folder C:\Office yang baru dibuat. Pastikan file setup.exe berada di folder ini.
Download Office Deployment Tool

Langkah 2: Konfigurasi (XML Hemat Resource)

Buat file bernama **Configuration.xml** di folder C:\Office. XML ini dikonfigurasi untuk hanya menginstal **Word, Excel, dan PowerPoint** untuk menghemat sumber daya dan ruang disk:

<Configuration> <Add OfficeClientEdition="64" Channel="PerpetualVL2021"> <Product ID="ProPlus2021Volume"> <Language ID="id-id" /> <ExcludeApp ID="Access" /> <ExcludeApp ID="Groove" /> <ExcludeApp ID="Lync" /> <ExcludeApp ID="OneNote" /> <ExcludeApp ID="Outlook" /> <ExcludeApp ID="Publisher" /> <ExcludeApp ID="Teams" /> </Product> </Add> <Display Level="None" AcceptEULA="TRUE" /> </Configuration>
(Pastikan nama filenya tetap **Configuration.xml**)
Opsi Advanced: Instalasi Office Penuh (Full Apps)

Untuk menginstal **semua aplikasi Office** (termasuk Outlook dan OneNote), gunakan XML berikut. Salin dan ganti isi Configuration.xml dengan kode ini:

<Configuration> <Add OfficeClientEdition="64" Channel="PerpetualVL2021"> <Product ID="ProPlus2021Volume"> <Language ID="id-id" /> </Product> </Add> <Display Level="None" AcceptEULA="TRUE" /> </Configuration>

Langkah 3: Eksekusi Perintah di CMD

Sekarang kita akan memerintahkan ODT untuk mendownload dan menginstal Office. **Wajib Buka Command Prompt sebagai Administrator**.

1. Masuk ke folder Office dengan perintah:

cd /d C:\Office\

2. Perintah Download (Mengunduh file instalasi ke folder):

setup.exe /download Configuration.xml
⚠️ Perhatian: Saat *download*, CMD terlihat kosong/diam (tidak ada progress bar). Itu normal. Biarkan jendela CMD tetap terbuka. Proses selesai jika kursor prompt kembali muncul.

3. Perintah Install (Menginstal Office dari file yang sudah diunduh):

setup.exe /configure Configuration.xml

✅ Instalasi selesai. Anda bisa menghapus folder C:\Office. Lanjutkan ke Bab 3 (MAS) untuk aktivasi Office.

Troubleshooting ODT

  • **Error 0-1018 (atau sejenisnya):** Biasanya terjadi karena konflik lisensi Office lama. Coba *uninstall* semua versi Office dan *reboot* sebelum mencoba lagi.
  • **Setup.exe langsung tertutup:** Kemungkinan besar Anda tidak menjalankan **Command Prompt sebagai Administrator** atau ada kesalahan pengetikan di perintah cd /d C:\Office\.
  • **Download stuck di 0% (tidak bergerak):** Koneksi internet tidak stabil, atau ada masalah *permission* di folder C:\Office. Coba jalankan perintah *download* dari folder lain (misalnya dari *Downloads*).
